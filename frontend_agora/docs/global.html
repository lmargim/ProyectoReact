generatePDF()

Genera un archivo PDF a partir del contenido de un elemento HTML con el ID "pdf-content". La función captura el contenido del elemento en un canvas utilizando html2canvas, convierte la imagen generada a formato PNG y la inserta en un documento PDF con jsPDF. Finalmente, guarda el archivo como "documento.pdf".
